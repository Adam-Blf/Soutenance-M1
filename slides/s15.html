C5.1  Cas d’usage GenAI

  • Besoin : créer une recette de cocktail originale à partir d’ingrédients disponibles.
  • Pourquoi GenAI : aucun moteur de règles ne peut produire une recette inédite cohérente.
  • Retrieval + génération : SBERT+FAISS pour le contexte, GPT-2 fine-tuné pour la recette.
  • Conformité : thématique alternative validée par l’Annexe I du sujet.

Pipeline

Ingrédients → SBERT → FAISS retrieval → guardrail 0.40 → cache MD5 → GPT-2 (recette)
